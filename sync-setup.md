# 药物记录跨设备同步设置指南

## 功能说明
现在您的药物记录应用支持跨设备同步功能，可以在多个设备间实时同步药物记录数据。

## 设置步骤

### 1. 获取 JSONBin API 密钥
1. 访问 https://jsonbin.io/
2. 注册免费账号
3. 登录后在控制台找到您的 API 密钥

### 2. 配置 API 密钥
在 `script.js` 文件中，将以下两处的 `YOUR_API_KEY_HERE` 替换为您的实际 API 密钥：

```javascript
'X-Master-Key': 'YOUR_API_KEY_HERE'
```

### 3. 使用同步功能
1. 在任意设备上打开药物记录应用
2. 点击右上角的"开启同步"按钮
3. 系统会自动创建云端存储并同步现有数据
4. 在其他设备上同样开启同步功能
5. 所有设备将自动保持数据同步

## 功能特点
- **自动同步**: 添加、编辑、删除药物时自动同步到云端
- **冲突处理**: 自动检测并处理数据冲突
- **离线支持**: 网络断开时仍可正常使用，恢复连接后自动同步
- **状态显示**: 实时显示同步状态

## 注意事项
- 首次同步需要有网络连接
- 所有设备需要使用相同的 API 密钥
- 建议定期备份重要数据

## 故障排除
如果同步失败，请检查：
1. 网络连接是否正常
2. API 密钥是否正确配置
3. 浏览器是否支持 localStorage

## 高级设置
您也可以考虑使用其他云存储服务，如：
- Firebase
- AWS S3
- 腾讯云 COS
- 阿里云 OSS

只需修改 `syncToCloud()` 和 `syncFromCloud()` 方法中的 API 调用即可。
